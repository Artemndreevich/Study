Які є стани гіперпосилань

Псевдокласси та робота з ними

Іноді звичайних селекторів недостатньо для визначення правила. Зазвичай псевдокласси потрібні для більш складної роботи правила. Псевдоклас такожж має незвичайну форму роботи : елемент:псевдоклас{правила}

Робота з гіперпосиланнями

Для гіперпосилань (<a>) достуні наступні псевдокласси

  • hover-- цей псевдокласс буде активуватись лише для тих посилань,на які наведео курсор миші
  • active-- буде активний лише при натисненні на посилання
  • focus-- активний лише при фокусуванні на посиланні
  • link-- активний на невідвіданих посиланнях
  • visited-- активний на відвіданих посиланнях
