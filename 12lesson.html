Власне шрифт

Для вибору шрифта використовують правило font-familу.

Після двокрапки вказується родина шрифтів або власне шрифт (бажано кілька шрифтів, на випадок якщо один або кілька з вказаних шритів не підтримується операційною системою).
p {
font-familу: Verdana, Geneva, Arial, sans-serif;
}

Всі шрифти поділяються на родини. Існує 5 родин шрифтів:

Назва родини Пояснення Приклади шрифтів
serif Шрифти з засічками. Стоїть по замовчуванню Cambria, Cochin, Georgia, Times, 'Times New Roman'
sans-serif Шрифти без засічок Verdana, Geneva, Tahoma, Trebushet MS, Arial
cursive Рукописні, курсивні шрифти Comic Sans MS, Bradley Hand, Brush Script MT, Snell Roundhand
fantasy Декоративні шрифти Impact, Luminari, Marker Felt, Trattatello
monospace Шрифти з фіксованою шириною символів Andale Mono, Courier New, Courier

Розмір шрифта

За розмір шрифта відповідає правило font-size

Розмір шрифта можна вказувати у пікселях (px), відсоткх (%) та у маштабній одиниці em. Значення em, як і %, показує скільки відсотків від початкового розміру буде становити новий розмір. Але це навідміну від %, число em поділене на 100. Тобто, 1em = 100% = початковому розміру шрифту. Тоді 2em, відповідно, збільшує шрифт удвічі (2em = 200%), 1.2em становитиме 120% від початкового шрифтy, а 0.5em зменшить шрифт удвічі (0.5em = 50% від початкового шрифту).

See the Pen ZEzjWag by rita (@RitaKr) on CodePen.

Як бачите, один і той же розмір шрифту можна задати різними способами. Але для цього потрібно знати початкові розміри шрифту різних тегів. Для цього ми приготували порівнялюну таблицю розмірів шрифтів у різних тегів по замовчуванню:

Назва тегу Розмір у px Розмір у % відносно <p> Розмір у em відносно <p>
<p> 16px 100% 1em
<h1> 32px 200% 2em
<h2> 24px 150% 1.5em
<h3> 19px 120% 1.2em
<h4> 16px 100% 1em
<h5> 14px 90% 0.9em
<h6> 10px 60% 0.6em

Значення еm та % буде залежати від розміру шрифту батьківсьокого елементу, або ж якщо шрифт для нього не вказаний, то від значення по замовчуванню. Наприклад, якщо для <body> ви вказали розмір шрифту 20px, а для <р> 0.9em, то розмір шрифту <р> буде становити 18px (20*0.9=18), адже CSS рахує розміру шрифту значення <р> відповідно до значення батьківського <body>. Якби ви не вказали font-size для <body>, то розмір шрифту рахувався б від значення по замовчувааню (16px для <р>)

Жирність шрифта

За жирність шрифту відповідає правило font-weight

Правило font-weight може приймати такі аргументи:

  • bold - напівжирний. Прописаний по замовчуванню для заголовків усіх рівнів
  • bolder - жирний
  • lighter - яскравий (тонкий)
  • normal - звичайний. Прописаний по замовчуванню для <p>

Стиль тексту

Для зміни стилю тексту використовують правило font-style

font-style впливає на стиль тексту. Існує 3 аргументи font-style:

  • normal - стоїть по замовчувааню і не змінює нічого
  • italic - робить текст курсивом (CSS аналог <em>)
  • oblique - робить текст похилим

Міжрядковий інтервал

Для змінення міжрядкового інтервалу використовують правило line-height

Іноді для звертання уваги треба зменшити або збільшити можрядковий інтервал. Цим керує правило line-height, воно може приймати такі аргументи:

  • normal - відверниту змінення до звичайного інтервалу
  • Значення у px
  • Значення у em
  • Значення у %

Декорація тексту

Для декореції тексту лініями використовують правило text-decoration

Правило text-decoration може приймати такі аргументи:

  • overline - лінія над текстом
  • underline - лінія під текстом (звичайне підкреслення)
  • wavy underline - хвилясте підкреслення
  • dashed underline - підкреслення пунктиром
  • dotted underline - підкреслення у крапочку
  • double underline - подвійне підкреслення
  • line-through - лінія перетинає текст по-середині
  • none - видалити всі декорації тексту

Вирівнювання тексту

Для вирывнювання тексту по-центру або по певному краю використовують правило text-align

Зазвичай, текст вирівнюжться по лівому краю, але це можна змінити за допомогою правила text-align та його аргументів:

  • left - вирівнює по лівому краю. Стоїть по замовчуванню
  • right - вирівнює по правому краю
  • center - вирівнює по центру
  • justify - заповнює весь рядок равномірно

Завдання:

Створіть сторінку-статтю на будь-яку тему, дотримуючись рекомендацій замовника:

  1. Шрифт усієї сторінки має бути без засічек
  2. Розмір шрифту усієї сторінки має становити 18px
  3. Розмір шрифту заголовків має бути в півтори рази більшим ніж по замовчуванню
  4. Заголовки мають бути вирівняними по центру
  5. Підзаголовки мають бути підкресленими хвилястою лінією і також вирівняними по центру
  6. Текст абзаців має заповнювати весь рядок
  7. інтервал між рядками у заголовків повинен становити 50%
