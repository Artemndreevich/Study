Рівні вкладеності

У цьому уроці ви дiзнаєтеся :

  • Що таке рівні вкладеності
  • Додаткові назви елементів

Про рівні вкладеності

Все логічно

Коле елемент (тег) лежить у іншому елементі,то він називається вкладеним. Тобто різні рівні цих єлементів й називаються рівнями вкладеностями. Наприклад візьмемо текст з минулого уроку

See the Pen mNxoKz by artem (@artemndreevich) on CodePen.

На 1 рівні вкладеності <ul> ,далі на другому <li>, на третьому <ol> і на четвертому - <li>. Логіка легка-чим менше рівень вкладеності тім більше елементів він обгортає.

Назви елементів

Елементі також бувать родинні,дочірні та материнські.

See the Pen mNxoKz by artem (@artemndreevich) on CodePen.

На цьому прикладі ми бачимо що <ul> - материнська для всіх єлементів,адже має найменший рівень вкладеності. <li>,який у <ul> - материнській для <ol> та для <li>,який у <оl>,родинний для <li>,який у <ul> й дочірній для <ul>.

Підсумуємо:

Для створення списку використовується тег <ul> та <ol>

Нумерованный список

  1. a
  2. b
  3. c
  4. d

Маркований список список

  • a
  • b
  • c
  • d
