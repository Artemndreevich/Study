Id та class елементів

У цьому уроці ви дiзнаєтеся :

  • Що таке айді та класи і для чого вони потрібні
  • Як задати елементу айді та клас
  • В чому різниця між ними

Що таке клас та айді елементу?

Як ви вже знаєте, в HTML існує безліч різних тегів, кожен з яких виконує свою функцію та надає певних властиостей тим чи іншим елементам контенту на сторінці. На одній сторінці зазвичай використовується велика кількість різних тегів для одного коду.
Візьмемо, наприклад, тег <p> (з минулих уроків ви знаєте, що він відповідає за створення абзаців). Уявімо, що в нас на сторінці є декілька абзаців, але нам треба виділити один з них за допомогою CSS. Якщо ми задамо властивості для <p>, то зміняться усі абзаци на сторінці, тож було б чудово якось виділити потрібний абзац або кілька абзаців, з якими ми будемо працювати. Саме для цього існують атрибути class та id, що надають елементу своє окреме ім'я, з яким легше працювати всередині коду.

Як присвоїти елементу клас чи айді?

<p class="name">
<p id="name">
У атрибутів class та id немає якихось конкретних значеннь, ви можете присвоїти їм будь-яке ім'я, що не відобразиться на самій сторінці. Хоча зачекайте, не будь-яке. В HTML існує кілька правил за якими треба давати назву класу чи айді (правила однакові для обох). Ми вже розглядали ці правила в уроці 4, але ми вам їх нагадаємо:

В чому різниця між класом та айді?

Призначення обох атрибутів однакове - задавати елементам додаткове ім'я. Але є невелика різниця.

Class Id
Class краще використовувати коли потрібно класифікувати певну группу елементів, тобто коли на сторінці є декілька елементів з однаковим ім'ям (класом). Class використовують, коли однакові CSS властивості треба присвоїти декільком різним елементам (Наприклад, зробити відступи у 20 пікселів зображенням, абзацам, заголовкам), або ж певним однаковим елементам (Наприклад, зробити текст перших трьох абзаців синім, при цьому не змінюючи колір тих, що нижче). Id ж варто використовувати, коли ви задаєте елементу унікальне ім'я, яке більше не буде повторюватись на цій сторінці. Це не означає, що задати айді можна лише одиному елементу на сторінці, ні. Айді можна задати і декільком елементам, але в такому випадку їм варто присвоїти різні ім'ена.

Підсумуємо:

  • Class і id - атрибути, які задають елементу додаткове ім'я, для подальших маніпуляцій у CSS. Приклади с уроку: <p class="name">, <p id="name">
  • Будь-якому елементу можна присвоїти клас або айді
  • Атрибутам сlass і id можна просвоїти будь-яке значення, яке відповідає правилам синтаксису HTML.
  • Class використовують для присвоєння спільного ім'я кільком елементам на сторінці.
  • Id використовують для присвоєння елементу унікального ім'я, яке більше не буде повторюватись.
