Вступ до CSS

Ви вже чудово знаєте, як спроектувати свою веб-сторінку: знайомі з базовими тегами, форматуванням, вмієте працювати з контентом, зображеннями та таблицями. Але лише технічної частини для створення вдалої веб-сторінки недостатньо. Велику роль також грає її зовнішній вигляд – красива веб-сторінка приваблюватиме більше користувачів. Звичайно, займатися її прикрашанням можна й через html, але мова CSS відкриває більше можливостей і є зручнішою у користуванні.

Отож, розпочнімо наше знайомство з мовою CSS!

Що таке CSS

CSS (Cascading Style Sheets, каскадні таблиці стилю) – формальна мова опису зовнішнього вигляду HTML-документу, за допомогою якої ви стилізуєте веб-сторінку, надаєте елементам красивого вигляду, розміщуєте їх на сторінці, тощо.

Як створити CSS-файл?

Розпочнімо з того, що стилізувати сторінку за допомогою CSS можна всередині HTML-файлу або в окремому CSS-файлі.

Щоб стилізувати сторінку всередині HTML-файлу, треба писати CSS-властивості в тезі <style>, який в свою чергу пишеться у вже відомому вам тезі <head>.

See the Pen poogeYB by rita (@RitaKr) on CodePen.

Але такий споосіб нераціональний при створенні сайту і незручний, тому ми будемо користуватися другим.
Для цього створюємо новий файл (у блокноті чи sublime text), та надаємо йому розширення .сss (згадайте, як ми робили це для HTML-файлу). В цьому файлі ми будемо створювати всю стилізацію сторінки.

Як під'єднати CSS-файл до HTML-файлу

Коли сторінка стилізації створена, обов’язково потрібно з’єднати її із самою веб-сторінкою з контентом. Для цього потрібно додати <link rel="stylesheet" href="назва файлу.css">, у <head> в HTML-файлі, що зв’яже два файли між собою.

Тег <link> використовується для того, щоб поєднати декілька сторінок, атрибут rel натякає на функцію, що виконуватиме підв’язана сторінка, коли атрибут href відповідає за її точну адресу для самого поєднання.
Зверніть увагу: тег <link> не потребує закриття!
<head>
<link rel="stylesheet" href="назва файлу.css">
</head>
Одну CSS-сторінку ви можете зв’язавти з багатьма веб-сторінками, але для цього потрібно бути обережним і слідкувати за виконанням коду.
Важливо: у самому коді CSS-документу з’єднати її з іншою сторінкою не можна.
Порада: називайте однаково веб-сторінку та її стилізацію для легшого маніпулювання ними. Наприклад: index.html та index.css.

Синтаксис CSS

Тепер почнемо говорити про наповнення та формат коду CSS.

Навідміну від сторінки у форматі HTML, тут не використовуються теги. Вам не потрібно використовувати жодних з них, для того щоб оформити вашу сторінку – CSS працює дещо інакше, хоча теги й згадуються.

CSS використовується для надання певних властивостей певним елементам коду HTML, і записується наступним чином.

Наведемо приклад, щоб потім розібрати його детально:
h1 {
color: red;
}
Цим кодом ми задали всім заголовкам першого рівня червоний колір. У даному прикладі h1 – селектор, тобто елемент, якому надається властивість.
Селекторами можуть бути теги (тобто контент вміщений всередині них), класи, айді і тд.

Після селектора відкривається фігурна дужка, і записується властивість (правило). В нашому випадку властивістю є color. Властивість – це певна характеристика, якої набуватиме елемент. Це може бути колір, розмір, прозорість, стиль тощо (з видами властивостей ви ознайомитесь пізніше).
Після правила ми ставимо двокрапку і задаємо значення властивості (аргумент) – колір, цифру, вид і тд, в залежності від самої властивості.
Зверніть увагу на розділовий знак крапки з комою, який обов’язково потрібно ставити після кожного значення властивості.
За такою схемою можна надати декілька властивостей одному елементу, чи декільком елементам одну властивість, написавши їх через кому.

Для зручності рекомендуємо записувати код CSS саме таким чином – властивості прописувати з нового рядка і через таб.

Псевдоелементи та Селетори-атрибути

Бувають випадки, коли ви хочете задати певну властивість лише частинці тегу чи елементу по всій сторінці – наприклад, першому рядку абзаца чи першій букві заголовку. В таких випадках можна вточнити селетор за допомогою псевдоелементів та селекторів-атрибутів.

Псевдоелементи

p:first-letter {
font-weight: bolder;
}
В цьому прикладі ми задаємо жирність шрифту (детальніше про цю властивість ви дізнаєтесь у наступних уроках про роботу з текстом) лише для перших букв всіх абзаців сторінки. Псевдоселектор в цьому випадку - first-letter. Існує ще й інший псевдоселектор - first-line, який стилізує перший рядок елементу.
Зверніть увагу, що псевдоелемент пишеться разом з селектором і відділяється від нього двокрапкою!

Селектори-атрибути

Також можна задавати властивості для тих елементів, що вже мають конкретні характеристики або атрибути, задані в HTML.
За приклад візьмемо зображення з шириною у 400px і задамо їм певну властивість:
іmg[width="400"] {
float: right;
}
Щоб задати таку деталізацію селектора, потрібно лише додати описову характеристику в квадратні дужки перед властивістю. Це може бути висота або ширина елемента, title, alt та будь-який інший атрибут що є в даного селектора.

Підсумуємо:

CSS - це формат, який дозвляє змінювати дані відповідно наданим характеристикам. Структура CSS – це перелік селекторів, тобто елементів, що ми змінюємо, та властивостей зі значеннями, яких ми задаємо (як у прикладі), що значно відрізняється від формату HTML.
